## 删表

```sql
Drop table if exists 表名；
```

—— 如果这个表存在的话，将这个表删除



## 删除数据

语法：

```sql
Delete from 表名 where 条件；
```

> [!TIP]
>
> 注意：没有条件全部删除

### 删除大表

```sql
Truncate table 表名；
```



`delete` 可以理解为橡皮，一行一行进行回滚删除，而 `truncate` 则是碎纸机，直接全部截断，全部消失【truncate 的效率比 delete 更高】
